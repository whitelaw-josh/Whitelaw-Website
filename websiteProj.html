Website

  • This website you are currently viewing was coded from scratch. I studied HTML and CSS coding at codecademy to better understand the basics. These basics include understanding spacing, formatting, and layout design.


    As I started programming the basics for the website I began to understand the importance of spacing and centering objects. This was especially critical if the user has a different screen size or changed the window size for their web browser. A proper color scheme to not overload the page with several colors and a pleasing font to the eye was also key in making a proper design.


    Lastly, I incorporated the method of Spriting with the some of the buttons in order to help with loading times. After making the first page (About Me), the layout of different div elements and alignment of them properly became much easier as I designed each of the projects, courses, and resume page.

  • One of my future intentions for this website is to incorporate animated elements using jAngular. I want to have the website look smoother as you transition through different pages and have it seamless. It would also help to animate the page buttons to make the page look more modern and professional.


    Another project would be to have a mobile layout of my website in mind for people looking at my website via a tablet or smart-phone.


    Finally, I would also like to incorporate a mailto element incorporating PHP to further my experience with the language.
